No Diagrama de fluxos abaixo representa os estados da maquina

Considerando que o aparelho pode queimar de tanto ficar utilizando foi setado um valor padrão máximo de utilização. Assim definido como se há ou não há a possibilidade de mudança. Afinal o fluxo precisaria de um fim.

Explicação Analogamente ao implementado e explicado em Link Maquina de Gomas podemos notar que IMaquinaState é análogo IAparelhoState, assim como o +solicitaGoma(int moeda) é a +solicitaTrocaEstado() que nos AparelhoState estão representados das seguintes formas:

  • no estado Desligado, por exemplo:

` public class Desligado extends AparelhoState{

public Desligado(Aparelho ap) {
    super(ap);
}

@Override
public void solicitaTrocarEstado() {
    this._aparelho.setNovoEstado(new Ligado(this._aparelho));
    this._aparelho.estado = true;
    this._aparelho.solicitaTrocarEstado();
}

` Análogo ao Estado Ligado

  • no estado Queimado

` public class Queimado extends AparelhoState{

public Queimado(Aparelho ap) {
    super(ap);

@Override
public void solicitaTrocarEstado() {

@Override

} `

Aplicação do Padrão Observador

Representação da Relação de Observação

No Diagrama abaixo está representada a relação entre o observador (o Aparelho) e o observado (o Controle Remoto Universal).

Uma relação que se pode notar quem emite o sinal de comando (o Controle Remoto Universal) e quem executa em si o comando pedido (o Aparelho).

Explicação

Analogamente ao implementado e explicado em Link ArCondicionado podemos notar que no Observable (nesse caso como Aparelho) contém funções de registro, delete e notificação de cada observer (que nesse caso, só é um, o Controle Remoto Universal(CRU)) que foram implementadas num formato análogo, só que com determinadas mudanças no processo, identificadas a seguir.

É dado o processo de controle seguindo esta sequência:

  • é feita a criação de um observer, sendo esse o CRU
  • se cria os aparelhos
  • se adiciona o CRU em cada um dos aparelhos
  • então se manipula as mudanças de estado, partindo de que o estado padrão é Desligado
